Spring/JPA приложение c авторизацией и правами доступа на основе ролей с использованием популярных инструментов и технологий Java: Maven, Spring Boot, Security, JPA(Hibernate), Bootstrap (css,js), datatables, jQuery, Fetch Api, Java 8 Stream и хранением в базах данных Postgresql.

Thymeleaf в качестве шаблонизатора (+thymeleaf-extras).
Конфигурация контекста при помощи анотаций.
Lombok для уменьшения шаблонного кода.
Cache(EhCache) на уровне Service для часто запрашиваеммых, но редко изменяемых ресурсов.
JUnit 5, Mockito для unit-тестов + MockMvc для постройки/выполнения запросов в тестах контроллеров(WebMvcTest) + Testcontainers (нужен Docker) для прогонки тестов репозиториев(DataJpaTest) и тестов системы в целом(SpringBootTest) на рельных БД.
